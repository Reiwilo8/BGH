# Known Limitations

---

## Text-to-Speech

- Quality and availability depend on system-installed voices
- No cloud-based voices are used
- macOS and Linux builds currently lack native TTS

---

## Platform Constraints

- Android minimum version dictated by Unity 6.0 LTS
- iOS testing requires physical devices for accurate behavior
- Desktop gesture input is simulated

---

## Gameplay Scope

- Only one fully implemented game (Memory)
- Other games are structural placeholders
- No competitive or score-based mechanics

---

## Visual Mode

- Visual Assist is intentionally minimal
- No advanced animations or complex UI layouts
- Focus remains on clarity and accessibility
