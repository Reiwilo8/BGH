# Games Overview

Blind Games Hub is designed as a modular platform capable of hosting multiple independent games.
Each game is implemented as a separate scene and module.

## Current Game Status

| Game ID     | Name          | Status        | Notes |
|------------|---------------|---------------|-------|
| memory     | Memory        | Complete      | Fully playable, tutorial, statistics implemented |
| steamrush  | Steam Rush    | Placeholder   | Scene and menu only, no gameplay |
| fishing    | Fishing       | Placeholder   | Scene and menu only, no gameplay |

---

## Memory

**Memory** is the first fully implemented game in the application.

Features:
- Multiple difficulty modes
- Tutorial mode
- Configurable board size
- Deterministic runs via seed system
- Persistent run statistics
- Fully audio-accessible gameplay
- Optional Visual Assist support

---

## Placeholder Games

**Steam Rush** and **Fishing** currently exist as structural placeholders.
They validate:
- Scene loading
- Game module lifecycle
- Integration with hub, settings, and run tracking

No gameplay logic is implemented for these games at this stage.
